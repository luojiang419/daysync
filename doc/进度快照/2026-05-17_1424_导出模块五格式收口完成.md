## 本次完成内容

1. 补全 `FCPXML` 导出能力：
   - 后端新增 `/api/projects/{project_id}/exports/fcpxml`
   - `daysync_core.export` 新增 `export_sync_report_fcpxml`
   - 输出文件扩展名使用 `.fcpxml`
2. `FCPXML` 按 Apple 官方文档的当前结构实现最小可导入版本：
   - 根元素 `fcpxml version="1.10"`
   - `resources` 中声明 `format` 与 `asset`
   - 用一个 `event` 承载多个 `project`
   - 每个 sync result 导出为一个单独 `project`
   - 每个 `project` 内有一个 `sequence` 和一个 `spine`
   - 通过 `asset-clip` + 嵌套 anchored `asset-clip` 表达视频/外录音频同步关系
   - 使用 `srcEnable="video"` / `srcEnable="audio"` 避免把参考音和外录音频混成同一个媒体组件
3. 前端导出页收口：
   - 新增“导出 FCPXML”按钮
   - 默认输出路径自动从 `sync_report.csv` 推导为 `sync_report.fcpxml`
   - 最近导出记录计数文案进一步细化：
     - `CSV`：`行`
     - `FCP 7 XML`：`sequence`
     - `FCPXML`：`project`
     - `JSON / OTIO`：`同步结果`
